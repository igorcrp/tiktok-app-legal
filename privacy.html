Esta Política de Privacidade explica como [SEU NOME/EMPRESA] (“nós”) coleta, usa e protege dados ao operar o aplicativo [NOME DO APP] (“Aplicativo”), que integra com a plataforma TikTok para automatizar publicação de conteúdo.

1. Quais dados coletamos e tratamos

  • Dados de autenticação do TikTok: tokens de acesso/refresh e permissões concedidas via OAuth (quando aplicável).
  • Dados da conta TikTok: identificadores básicos necessários para publicar (por exemplo, ID da conta/usuário fornecido pela API).
  • Dados do conteúdo: metadados do vídeo (título/legenda, configurações de publicação) e arquivos enviados para publicação, quando você solicita a postagem.
  • Logs técnicos: registros de execução/erros para diagnóstico (ex.: horário, status, mensagens de erro).

2. Para que usamos os dados

  • Conectar sua conta TikTok ao Aplicativo e publicar vídeos automaticamente conforme sua solicitação.
  • Operar e manter o serviço, corrigir problemas e melhorar estabilidade.
  • Atender exigências de segurança e prevenção a abuso.

3. Base legal (LGPD, se aplicável)

Tratamos dados principalmente para execução de contrato (fornecer a automação solicitada) e/ou legítimo interesse (segurança, prevenção a fraude, melhorias), conforme aplicável.

4. Compartilhamento de dados

Não vendemos seus dados. Podemos compartilhar dados apenas:

  • Com o TikTok, por meio das APIs oficiais, para realizar as ações que você solicitou (ex.: publicar conteúdo).
  • Com provedores técnicos envolvidos na hospedagem/execução da automação (por exemplo, onde o n8n estiver rodando), estritamente para operar o serviço.

5. Onde os dados ficam armazenados

As credenciais/tokens podem ser armazenados no ambiente onde o fluxo do n8n estiver instalado, conforme sua configuração (ex.: servidor próprio, VPS, serviço gerenciado). Recomendamos uso de criptografia e controle de acesso adequado.

6. Retenção e exclusão

  • Tokens e dados técnicos são mantidos apenas pelo tempo necessário para operar o serviço.
  • Você pode revogar o acesso do Aplicativo à sua conta TikTok a qualquer momento nas configurações do TikTok (quando aplicável).

7. Segurança

Adotamos medidas razoáveis para proteger dados contra acesso não autorizado. Ainda assim, nenhum sistema é 100% seguro.

8. Seus direitos

Você pode solicitar informações, correção ou exclusão de dados conforme a legislação aplicável.
